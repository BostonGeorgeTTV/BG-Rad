# BG-Rad

BG Rad è uno script premium per FiveM pensato per server survival, post-apocalittici, military RP e roleplay con aree contaminate.

Il sistema permette di creare zone radioattive configurabili tramite ox_lib Poly Zones. Quando un player entra in un’area contaminata, la UI NUI dedicata si attiva mostrando il livello di radiazioni, lo stato di pericolo e le protezioni indossate.

Lo script include una UI moderna in stile survival post-apocalittico, effetti visivi, suono ambientale .ogg in loop dentro le zone radioattive e danni progressivi quando la contaminazione diventa critica.

BG Rad supporta anche equipaggiamento protettivo: la tuta antiradiazioni attenua la contaminazione, mentre la maschera antiradiazioni funziona tramite filtri intercambiabili. Ogni filtro viene consumato all’utilizzo e dura 5 minuti, scalati solo mentre il player si trova in una zona radioattiva.

È incluso un item siringa antiradiazioni per ridurre il livello di contaminazione acquisito. I livelli di radiazione e lo stato del filtro sono persistenti tramite database, quindi restano salvati anche dopo quit, crash o riavvii del server.

Caratteristiche principali:

Zone radioattive configurabili con ox_lib
UI NUI animata in stile post-apocalittico
Suono ambientale in area contaminata
Sistema radiazioni con danno progressivo
Tuta antiradiazioni con riduzione contaminazione
Maschera antiradiazioni con filtri da 5 minuti
Timer filtro scalato solo in zona radioattiva
Siringa antiradiazioni utilizzabile
Salvataggio persistente su database
Supporto multi-zona
Config semplice e ottimizzata

Dipendenze: ox_lib, ox_inventory, oxmysql.

Perfetto per bunker, laboratori, wasteland, eventi PvE, aree militari, zone infette, mappe custom e server survival RP.
